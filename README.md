# Parcial – Desarrollo Web (HTML + CSS)

Para este parcial, cada estudiante deberá elegir **una** de las dos opciones de diseño y **replicar el formulario únicamente con HTML y CSS**, sin usar frameworks ni JavaScript.

---

### Constellations (Desktop)

Repositorio: `constellations`  
Diseño en Figma: [Constellations – Contact Form (Desktop)](https://www.figma.com/design/laur81jfWyQEHpMoQ5qNFa/Contact-Form---Page--Community-?node-id=0-1&t=6UYRp2rKZyZQgcRi-1)

En esta opción deberás **maquetar en HTML y CSS la página de formulario de contacto “Constellations” para vista escritorio (desktop)**.

El objetivo es recrear con precisión el diseño de Figma, incluyendo:

- Distribución de la información en columnas y secciones  
- Estilos del formulario (inputs, botones, estados básicos)  
- Uso correcto de tipografía, tamaños y espaciados  
- Composición limpia y equilibrada pensada para pantallas grandes

> Restricciones:  
> - Solo se permite **HTML + CSS puro**  
> - No se pueden usar frameworks (Bootstrap, Tailwind, etc.)  
> - No se debe usar JavaScript
